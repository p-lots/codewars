Write a function `sum` that accepts an unlimited number of integer arguments, and adds all of them together.

The function should reject any arguments that are not **integers**, and sum the remaining integers.

```ruby
sum(1, 2, 3) # => 6
sum(1, '2', 3) # => 4
```
```python
sum(1, 2, 3)    ==>  6
sum(1, "2", 3)  ==>  4
```
```javascript
sum(1, 2, 3)   // -> 6
sum(1, "2", 3) // -> 4
```
