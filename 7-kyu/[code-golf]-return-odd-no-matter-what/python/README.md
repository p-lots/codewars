Given the integer `n` return odd numbers as they are, but subtract 1 from even numbers.

**Note:** 
~~~if:python,ruby
Your solution should be 36 or less characters long.
~~~
~~~if:javascript
Your solution should be 22 or less characters long.
~~~

### Examples

```
Input  = 2
Output = 1

Input  = 13
Output = 13

Input  = 46
Output = 45
```
