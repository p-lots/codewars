Write a program that outputs the `n` largest elements from a list.

Example:
```python
largest(2, [7,6,5,4,3,2,1])
# => [6,7]
```
```javascript
largest(2, [7,6,5,4,3,2,1])
// => [6,7]
```
```haskell
largest 2 [7,6,5,4,3,2,1]
-- => [6,7]
```
```csharp
Kata.Largest(2, new List<int> {7, 6, 5, 4, 3, 2, 1}) => new List<int> {6, 7}
```
